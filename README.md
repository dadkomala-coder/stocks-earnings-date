# Stocks Earnings Date Highlighter

Script that shows earnings dates for selected stocks and highlights upcoming earnings events.

## Description

This script displays the earnings dates of your favorite stocks.
If the earnings date is less than *31 days from the current date, the script highlights it **in blue* so upcoming reports are easy to notice.

This helps investors quickly see which companies are about to report earnings.

## Features

* Track earnings dates for selected stocks
* Automatic comparison with the current date
* Highlight earnings that occur within *31 days*
* Simple and lightweight script

## Usage

1. Add the stock tickers you want to track.
2. Run the script.
3. The script will display earnings dates.
4. Earnings dates within *31 days* will appear highlighted *in blue*.

## Example

| Stock | Earnings Date | Status                       |
| ----- | ------------- | ---------------------------- |
| AAPL  | 2026-04-25    | highlighted (within 31 days) |
| MSFT  | 2026-06-10    | normal                       |

## Purpose

Earnings announcements often cause strong market movements.
This tool makes it easier to track upcoming earnings events for stocks you follow.
